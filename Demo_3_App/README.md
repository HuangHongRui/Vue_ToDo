`<input type="" v-model="newTodo">`
这一句将 input.value 与 data.newTodo 绑定起来了，而且是双向的：
1. 只要 input.value 被用户改了，data.newTodo 就会变成一样的值；
2. 只要 data.newTodo 被 JS 改了，input.value 就会变成一样的值。
